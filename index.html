Think Webapp. Think JavaScript.

d3.js a powerful JS library is used.
d3 can manipulate DOM and data and eases the task of handling html elements and their corresponding data elements thereby giving user ample time to use their data in a most effective manner.

SVG

Scalable Vector Graphics. Used to draw the chart. d3 comes with a lot of APIs to make SVGs from different data sets.

The D3 magic

D3 comes bundled with API to make chord diagrams which include data processor and SVG generator

Our API only uses the SVG generator as our data processing requirements vary from the ones provided by d3. Hence we use our own data processors and format the data to be fed to these API.

How does the CircosJS API work?

Code


    circos.generate({
        'circosInfo':{
            'zoom':[1,10],
            'scaleInfo':{'scale':1,'stepAngle':1.8,'labelGap':5},
            'chordData':crdata,
            'arcData':ardata
        }
    });

This is the minimum code you'll need to write to get your first chart boilerplate.

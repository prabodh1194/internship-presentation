SVG

Scalable Vector Graphics. Used to draw the chart. d3 comes with a lot of APIs to make SVGs from different data sets.

The D3 magic

D3 comes bundled with API to make chord diagrams which include data processor and SVG generator

Our API only uses the SVG generator as our data processing requirements vary from the ones provided by d3. Hence we use our own data processors and format the data to be fed to these API.
